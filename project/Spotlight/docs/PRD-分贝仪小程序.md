## 2. 背景与目标

### 2.1 问题陈述

| 用户痛点 | 场景描述 |
|---------|---------|
| 学习环境噪音不可知 | 家长想确保孩子学习环境安静，但无法量化噪音水平 |
| 租房噪音难以评估 | 看房时想快速判断周边噪音情况，缺乏工具 |
| 办公/会议环境监测 | 需要安静环境的工作者（录音、直播、专注工作）想实时了解噪音状况 |
| 噪音超标无感知 | 噪音逐渐增大时用户不易察觉，影响睡眠或工作 |

### 2.2 业务目标

| 目标类型 | 指标 | 说明 |
|---------|------|------|
| 用户增长 | 7日留存率 > 25% | 工具类产品核心指标 |
| 用户活跃 | 人均使用时长 > 2分钟 | 单次测量场景 |
| 功能渗透 | 阈值设置率 > 40% | 核心差异化功能使用率 |

### 2.3 成功标准

- ✅ MVP 版本上线后 30 天内，日活用户 > 1000
- ✅ 用户评分 > 4.0 分（微信小程序评分）
- ✅ 无重大 Bug 反馈（崩溃率 < 0.1%）

### 4.2 详细功能规格

#### 功能 1：实时分贝测量

| 项目 | 内容 |
|------|------|
| **用户故事** | As a 用户, I want 实时看到当前环境分贝数值, So that 我能快速了解噪音水平 |
| **优先级** | P0 |
| **功能描述** | 通过麦克风采集环境声音，实时计算并显示分贝值，刷新频率 100ms |

**验收标准（Given-When-Then）：**

```gherkin
Given 用户已授权麦克风权限
When 用户进入主页面
Then 页面实时显示当前环境分贝数值（0-120dB）
And 数值每 100ms 更新一次
And 显示当前噪音等级描述（安静/正常/较吵/嘈杂）

Given 用户未授权麦克风权限
When 用户进入主页面
Then 显示授权引导弹窗
And 点击"授权"按钮触发系统授权弹窗

Given 用户拒绝授权
When 授权失败
Then 显示"无法使用，请授权麦克风权限"提示
And 提供"重新授权"按钮引导用户去设置页开启权限
```

**分贝等级对照表：**

| 分贝范围 | 等级 | 颜色 | 说明 |
|---------|------|------|------|
| 0-30 dB | 极静 | 🟢 深绿 | 图书馆级别 |
| 30-50 dB | 安静 | 🟢 浅绿 | 正常交谈 |
| 50-70 dB | 正常 | 🟡 黄色 | 办公环境 |
| 70-90 dB | 较吵 | 🟠 橙色 | 繁忙街道 |
| 90-120 dB | 嘈杂 | 🔴 红色 | 警报级别 |

---

#### 功能 2：分贝可视化展示

| 项目 | 内容 |
|------|------|
| **用户故事** | As a 用户, I want 直观看到分贝变化曲线, So that 我能更清晰地感知噪音波动 |
| **优先级** | P0 |
| **功能描述** | 实时波形图/柱状图展示分贝变化，视觉化反馈 |

**验收标准：**

```gherkin
Given 用户正在测量分贝
When 分贝数值变化
Then 波形图/柱状图实时跟随变化
And 当前分贝数值用大号字体突出显示
And 背景颜色根据分贝等级动态变化（绿→黄→橙→红）
```

---

#### 功能 3：阈值提醒设置

| 项目 | 内容 |
|------|------|
| **用户故事** | As a 用户, I want 设置分贝阈值, So that 当噪音超标时能及时提醒我 |
| **优先级** | P0 |
| **功能描述** | 用户可设置目标分贝阈值，当实时分贝超过阈值时触发提醒 |

**验收标准：**

```gherkin
Given 用户进入设置页
When 用户设置阈值（如 50dB）
Then 保存阈值设置到本地
And 返回主页后显示当前阈值指示线

Given 用户已设置阈值 50dB
When 实时分贝超过 50dB
Then 触发提醒（震动 + 音效 + 屏幕红色闪烁）
And 提醒间隔至少 3 秒（避免频繁触发）

Given 用户关闭提醒开关
When 实时分贝超过阈值
Then 不触发任何提醒
```

**提醒方式说明：**

| 提醒方式 | 实现方式 | 用户体验 |
|---------|---------|---------|
| 震动 | `wx.vibrateShort()` | 短震动一次 |
| 音效 | `wx.createInnerAudioContext()` | 播放"叮"提示音 |
| 视觉 | CSS 动画 | 屏幕红色闪烁 2 次 |

## 10. 附录

### 10.1 技术方案要点

#### 分贝计算公式

```javascript
// 简化分贝计算公式（基于音频振幅）
function calculateDecibel(rms) {
  // 参考：20 * log10(RMS / 参考值)
  const ref = 0.00002; // 参考声压（20微帕）
  const db = 20 * Math.log10(rms / ref);
  return Math.max(0, Math.min(120, db)); // 限制在 0-120dB
}
```

#### 关键 API

| API | 用途 |
|-----|------|
| `wx.getRecorderManager()` | 获取录音管理器 |
| `RecorderManager.onFrameRecorded()` | 实时获取音频帧数据 |
| `wx.vibrateShort()` | 短震动提醒 |
| `wx.createInnerAudioContext()` | 播放音效 |
| `wx.setStorage()` | 本地存储设置 |

#### 权限配置（app.json）

```json
{
  "permission": {
    "scope.record": {
      "desc": "需要使用麦克风来测量环境声音分贝"
    }
  }
}
```
