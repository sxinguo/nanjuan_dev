# 剪映字幕纠错技能

## 技能目标

对剪映（CapCut）自动语音识别（ASR）生成的字幕进行语义纠错，修正同音字/近音字错误、断句问题、冗余语气词等，同时**严格保留所有时间戳不变**，使纠错后的字幕文件可直接导回剪映使用。

---

## 支持的字幕格式

| 格式 | 扩展名 | 说明 |
|------|--------|------|
| SRT  | `.srt` | 最常见，剪映默认导出格式 |
| VTT  | `.vtt` | WebVTT 格式 |
| LRC  | `.lrc` | 歌词格式（含时间戳） |
| 纯文本 | `.txt` | 用户直接粘贴字幕文本 |

---

## 操作流程

### Step 1：识别输入格式

判断用户提供的是：
- **文件上传**：读取文件内容
- **直接粘贴文本**：直接处理
- **部分片段**：先确认是否为完整字幕，或只需处理片段

### Step 2：解析字幕结构

以 SRT 为例，解析每个字幕块：
```
[序号]
[开始时间] --> [结束时间]
[字幕文本]
```

**关键原则：时间戳行完全不得修改。**

### Step 3：语义纠错（核心步骤）

按以下优先级逐条检查字幕文本：

#### 3.1 同音字 / 近音字错误（最常见）
- 典型例子：
  - "在" ↔ "再"、"的" ↔ "地" ↔ "得"、"他" ↔ "她"
  - "必须" → "必需"（语义不符时）
  - "原来" → "缘来"（错误）
  - "事情" → "是情"（错误）
- **纠错方法**：结合前后句语义判断正确用字

#### 3.2 专有名词 / 术语识别错误
- 人名、地名、品牌名、行业术语容易被识别错
- 例：说的是"Claude"，识别成"Cloud"；说的是"抖音"识别成"豆音"
- **纠错方法**：结合上下文语境还原正确词汇
- 若无法确定，在该处添加注释 `[待确认]`

#### 3.3 断句错误
- ASR 有时在错误的位置断句，导致语义割裂
- 例：
  - 错误：`我觉得` / `这件事很重要`
  - 正确：`我觉得这件事` / `很重要`
- **注意**：调整断句时，**不能改变时间戳**，只调整文字在各时间段的归属（如确实需要，可合并相邻行再重新分配，但须说明）

#### 3.4 冗余语气词 / 口语填充词
- "那个"、"就是"、"然后"、"嗯"、"啊"、"对对对"等高频出现时
- **策略**：
  - 少量保留（保持说话风格）
  - 明显影响阅读体验时删除
  - **不主动删除**，除非用户明确要求精简

#### 3.5 标点符号规范
- ASR 通常不生成标点，或标点不规范
- 根据语义添加：句号、逗号、问号、感叹号
- 避免过度添加标点影响字幕节奏

### Step 4：输出纠错结果

#### 4.1 输出格式
按原始格式输出完整纠错后的字幕文件，格式与输入保持一致（SRT 输出 SRT，VTT 输出 VTT）。

#### 4.2 附上修改说明
在字幕文件之后，提供一份简洁的**修改日志**，格式如下：

```
【修改日志】
共修改 X 处：

第3行：「事情」→「是情」（同音字错误）
第7行：「再」→「在」（语义纠错）
第12行：[待确认] 疑似专有名词，已标注
...
```

#### 4.3 不确定项处理
- 对无法确定的词，在原文中用 `[待确认: 原文]` 标注
- 在修改日志中单独列出所有待确认项，方便用户复查

### Step 5：用户确认与迭代

- 若用户反馈某处纠错有误，重新修正该处并更新日志
- 若用户希望进一步精简语气词，重新输出精简版本
- 最终版本以**干净的字幕文件**形式输出，供用户导回剪映

---

## 注意事项

1. **时间戳绝对不能修改**：剪映依赖时间戳对齐视频，任何时间戳变动都会导致字幕错位。
2. **保守纠错原则**：只在有把握时修改，不确定的用 `[待确认]` 标注。
3. **保留说话风格**：纠错目的是修正错字，不是重写字幕，尽量保留原说话者的表达习惯。
4. **逐块处理**：每个字幕块独立处理，同时参考相邻块的上下文语义。
5. **长字幕分批处理**：若字幕超过 200 条，告知用户将分批处理，并逐批输出。

## 常见剪映 ASR 错误模式速查

| 错误类型 | 典型例子 | 纠正思路 |
|----------|----------|----------|
| 同音字 | 在/再、的/地/得、做/作 | 看词性和语境 |
| 数字混淆 | "一百"识别成"一摆" | 结合上下文数量词 |
| 叠词拆分 | "刚刚"识别成"刚 刚" | 合并 |
| 专有名词 | 人名/品牌识别成常见字 | 标注待确认 |
| 句末吞字 | 最后一个字丢失 | 根据语义补全 |
| 方言词汇 | 普通话中混入方言音 | 结合语境还原标准词 |
