# 微信公众号图片消息草稿创建 Skill

## 功能说明

快速创建微信公众号图片消息草稿（公众号后台"草稿箱"里的图片消息）。

**使用流程：**
1. 用户提供：标题 + 描述信息 + 图片（至少1张）
2. 自动上传图片为永久素材
3. 自动获取/刷新 access_token
4. 调用草稿 API 创建图片消息草稿
5. 返回结果

## 首次配置

首次使用需要配置公众号凭证：

```
AppID:  你的公众号 AppID
AppSecret: 你的公众号 AppSecret
```

请告诉我你的 AppID 和 AppSecret，我会帮你保存配置。

## 使用方式

**触发词：**「创建草稿」「新建图片草稿」「创建图片消息草稿」

**输入格式：**
- 标题：（标题内容，最多32个字）
- 描述信息：（正文内容/描述，最多128个字）
- 图片：（发送图片文件，至少1张，最多20张）

## API 端点

| 功能 | API |
|------|-----|
| 获取 token | `GET https://api.weixin.qq.com/cgi-bin/token` |
| 上传永久图片素材 | `POST https://api.weixin.qq.com/cgi-bin/material/add_material?type=image` |
| 创建草稿 | `POST https://api.weixin.qq.com/cgi-bin/draft/add` |

## 错误处理

- Token 过期：自动刷新
- 图片上传失败：提示用户重试
- 草稿创建失败：显示具体错误信息

## 注意事项

- 图片建议使用 jpg/png 格式
- 图片会上传为永久素材（有数量限制，当前 10000 个）
- 首张图片会自动作为封面图
- 标题最多 32 个字，描述最多 128 个字
